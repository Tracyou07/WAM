Core Contrast

三个关键区分

World Model

建模世界

关注给定动作后环境如何变化，常用于未来预测、模拟展开和 model-based planning。

WAM

为动作建模世界

把未来世界和动作放进同一个控制问题中，目标是让世界表征服务于动作生成。

Fast-WAM

训练时想象，推理时行动

保留视频预测作为表征学习约束，但测试时跳过显式未来视频生成以降低延迟。
